# Lifeline TTS

## How to use

1. Clone this repo.
2. Create a `.env` file
3. In the `.env` file put the following:

```dotenv
AWS_ACCESS_KEY_ID={YOUR_KEY}
AWS_SECRET_ACCESS_KEY={YOUR_SECRET}
```

4. Install all the python packages with `pip install -r requirements.txt`
5. Run the code with `python3 main.py`
